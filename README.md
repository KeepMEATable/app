# KeepMeATable - Mobile App

This app is made with react native through expo

It has the following modules installed:
* redux
* react-redux
* react-navigation
* react-native-gesture-handler


## installation

```
$ yarn install
$ yarn start
```

specific android

```
$ yarn android
```

specific ios

```
$ yarn ios
```

## build

```
$ yarn eject
```
